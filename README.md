# General Information 

This is the repository for my final assignment for ENVS 193DS. The project contains answers to three parts showcasing my statistical knowledge and capabilities to assess, analyze, and visualize data. Part 1 focuses on a statistical critique, Part 2 focuses on assess and analyzing nesting occurence for birds based on tree height and acacia ant species type, while Part 3 focuses on further analyzing personal health data related to sleep duration and total daily activity level.
